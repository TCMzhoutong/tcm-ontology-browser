# 中医术语词表浏览器

这是一个静态网页版本的中医术语词表浏览器。

- 入口文件：`index.html`
- 数据文件：`data/*.json`
- 站点可通过 GitHub Pages 直接访问

## 在线访问

https://tcmzhoutong.github.io/tcm-ontology-browser/
